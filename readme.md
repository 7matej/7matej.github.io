# Můj blog
Cílem tohoto projektu je vytvořit webovou stránku se pokud možno zajímavými články o různých tématech, která mě zajímají a ke kterým mám co říct například matematika, fyzika, cestování ... Chci, aby byl obsah co nejvíce originální a doufám, že tyto stránky si najdou své čtenáře. Autorem tohoto projektu je Matěj Klíma.

## Aktuální stav
Zatím je rozdělaná první stránka o zrcadlech ve tvaru kuželoseček. Upozornění: Stránka index.html je totální balast vytvořený umělou inteligencí. V žádném případě ji prosím neberte vážně. Zatím ji zde ponechávám ze školních důvodů (hodí se na různé experimenty a domácí úkoly).
